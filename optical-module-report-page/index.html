三、国内光模块 vs 海外光模块：不是简单替代关系

维度中国龙头：中际/新易盛/天孚/光迅海外龙头：Coherent、Lumentum、Corning、AAOI、Fabrinet等投资判断
核心优势工程迭代快、制造成本强、交付弹性大，在800G/1.6T模块侧具备全球竞争力；中际和新易盛更接近“模块制造+客户认证”龙头。在激光器、光子芯片、外置光源、光纤连接、硅光/CPO生态和美国本土供应链中更有战略位置。中国公司强在模块量产和成本，海外公司强在上游光子器件、客户合规和AI巨头战略绑定。
客户结构海外云厂商贡献关键增长，但也带来地缘和合规风险；国内算力建设提供第二增长曲线但利润率需验证。与Nvidia、北美云厂商、网络设备商关系更稳，部分公司获得投资或采购承诺。美国客户若“去风险化”，国内龙头估值需折价；若订单正常，国内龙头盈利弹性更强。
技术路线800G/1.6T可插拔模块为主，硅光、CPO/NPO均有布局；天孚在CPO配套器件更突出。Lumentum、Coherent等在激光、光子组件、外置光源、OCS/CPO方案中更靠近底层技术。未来如果CPO/外置光源占比提升，利润可能向光子器件和平台型供应商转移。
政策风险中际旭创被列入美国1260H清单虽非直接出口禁令，但会影响估值和客户合规审查。受美国本土制造、Nvidia投资、供应链安全偏好支持，但成本更高。国内股需要用更高盈利增速补偿政策折价。
估值状态A股龙头短期重估极快，光迅等二线也被显著拔估值。海外光网络股2026年同样强势，海外媒体持续提示AI光网络交易很热。全球都不便宜，不能把海外热度简单外推为A股继续扩张空间。
国内外比较结论：2026年最确定的利润仍在800G/1.6T交付，国内龙头有成本和规模优势；但2027年后若CPO、外置光源、硅光平台化加速，海外光子器件和材料平台的战略溢价可能更高。A股组合应把“模块龙头”与“器件/材料平台”分开估值。

哪一环更值得看？按兑现确定性排序

排序环节为什么风险组合动作
1AI PCB / 高速交换机PCBAI服务器、GPU板、交换机和高速背板直接提升层数、材料等级和加工难度；订单与收入传导最快。胜宏、深南、沪电等估值已高，若AI服务器资本开支放缓，利润和估值双杀。核心延伸仓优先等回撤买沪电/胜宏/深南。
2覆铜板/高频高速材料高速PCB升级带动低损耗材料、树脂体系、铜箔和玻纤布需求；生益等具备材料卡位。涨价周期容易吸引新产能，毛利率能否持续需跟踪。择机配置不在极高PE时追。
3MLCC/陶瓷元件AI服务器电源和高速板提升高可靠被动元件用量；国产替代逻辑存在。行业仍有消费电子和通用周期属性，AI纯度低于PCB。卫星仓更适合等待业绩拐点。
4电子布/电子纱高端覆铜板上游，供需紧时价格弹性大。多数公司AI收入占比难直接穿透，股价更容易由涨价预期驱动。交易仓用价格和库存信号，不做长期高估值重仓。
5玻璃基板/玻璃核心基板先进封装未来方向，海外Intel、Rapidus等推动玻璃基板和面板级封装探索。量产在早期，A股多数仍是技术期权，盈利兑现慢。小仓观察只买研发里程碑，不买宏大叙事。
